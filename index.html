W3C-valide XHTML Tabelle

Die Aufgabenstellung ist eine Webseite in gültiger XHTML 1.1 Syntax zu erstellen welche eine Tabelle mit allen 77 XHTML Elementen enthält. Diese Seite soll CSS 2(.1) kompatibel sein und Icon-Links zu den Validatoren für XHTML und CSS des W3C enthalten um die geforderte Kompatibilität nachzuweisen
